erd를 프로젝트 도중 끊임없이 수정하는 일이 발생했습니다. db에 컬럼이 없어서 데이터를 저장못하거나, 관계가 잘못 연결되어있어 fk를 다시 연결하거나 했습니다. 그러면서 erd를 제대로 작성하지 않으면 이후 코드작업 시에 끊임없는 수정사항이 발생한다는 점을 익혔습니다.crud를 실제 프로젝트에서 구현해보면서 crud에 대한 개념에 대해 더욱 정확하게 알 수 있게 되었습니다.
